Description of the project - QuickCalc

QuickCalc is a simple calculator tool to do simple maths as addition division or multiplication. It's developpend in javascript and uses the P5 library.

Using QuickCalc

To use QuickCalc first run 'npm install' and then simply open the file 'index.html' in the browser of your choice (e.g if you use firefox you can execute 'firefox index.html')

Running the tests

After having installed the dependencies with 'npm install' you can execute the testsuite with 'npm test'
